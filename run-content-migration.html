🎨 Content Management System Migration

Bu migration anasayfadaki tüm içerikleri yönetmek için database tabloları oluşturur

⚠️ Önemli Not

Bu migration'ı Supabase Dashboard'dan SQL Editor kullanarak çalıştırmalısınız.

Dosya: supabase/migrations/20251002140000_create_content_management_tables.sql

Migration İçeriği:

  • ✅ site_content - Hero, CTA, genel metinler
  • ✅ portfolio_items - Referans çalışmaları (önce/sonra görselleri)
  • ✅ testimonials - Müşteri yorumları
  • ✅ site_stats - İstatistikler (50,000+ Görsel, vb.)
  • ✅ features - Özellik kartları

Admin Paneli Yeni Sayfalar:

  • 📝 /admin/content - Genel içerik metinleri
  • 🖼️ /admin/portfolio - Referans çalışmaları yönetimi
  • ⭐ /admin/testimonials - Müşteri yorumları yönetimi
